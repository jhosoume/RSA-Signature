# Digital Signature Using RSA-OAEP and SHA-3

Recommended installation:
* Creating Python Env
```console
python3 -m venv rsa_sig
```

* Use the Python Env
```console
source rsa_sig/bin/activate
```

*  Get Required Packages
```console
pip install -r requirements.txt
```

* Run
```console
./pyssl
```

* To run a sample test
```console
./run_t1.sh
```
